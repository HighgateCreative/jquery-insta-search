# jQuery Insta-Search

## Usage

1. Include jQuery:

	```html
	<script src="http://ajax.googleapis.com/ajax/libs/jquery/2.0.0/jquery.min.js"></script>
	```

2. Include Insta-Search's code:

	```html
	<script src="dist/jquery.insta-search.min.js"></script>
	```

3. Call Insta-Search:

	```javascript
	$("#searchForm").instaSearch();
	```
